About Me

I'm a full-Stack Web Developer based in Argentina, focused on MVC .NET development. It is my thinking that code should be built clean, beautiful, organized and efficient (and well documented). Honestly, I like to follow good practices and style, and get things done on time.

About This Blog

My goal with this blog is to practice my Web development skills, mainly regarding the good, correct and best use of the very core technologies of it. All of this while using HTML, CSS, JS, and whatever I may find and use at the building of this little blog. As you may guess that way I'm accomplishing the double fold purpose of writing about Web Development while doing Web Development.
